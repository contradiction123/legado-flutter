## 4.2 详细步骤

### 步骤 4.1：EPUB 阅读器

**做什么**：
- 实现 EPUB 文件解析（解压、解析 OPF、获取章节）
- 实现 HTML 内容渲染（CSS 样式处理）
- 实现 EPUB 特有配置（字体嵌入、CSS 覆写）

**参考的原项目代码**：
- `legado-with-MD3/app/src/main/java/io/legado/app/model/localBook/EpubFile.kt` — EPUB 解析核心
- `legado-with-MD3/modules/book/` — 独立书模块（EPUB/UMD 解析库）
  - 命名空间 `me.ag2s`
  - 包含 EPUB 的 OPF 解析、NCX 目录解析等
- `legado-with-MD3/app/src/main/java/io/legado/app/ui/book/read/page/entities/TextHtmlColumn.kt` — HTML 列排版
- `legado-with-MD3/app/src/main/java/io/legado/app/ui/book/read/page/provider/TextChapterLayout.kt` — 章节排版引擎

**Flutter 重写内容**：
- 使用 `archive` 包解压 EPUB（本质是 ZIP 文件）
- 使用 `xml` 包解析 OPF 文件（获取元数据 + 章节列表）
- 使用 `html` 包解析 HTML 章节内容
- 使用 `flutter_widget_from_html` 或 `flutter_html` 包渲染 HTML 内容
- 创建 `lib/engine/local_book/epub_parser.dart` — EPUB 解析器
- 创建 `lib/features/reader/renderer/epub_renderer.dart` — EPUB 渲染器

**当前步骤实现的功能**：
- ✅ EPUB 文件读取和解压
- ✅ OPF 元数据解析（书名、作者、封面）
- ✅ 目录解析
- ✅ HTML 章节内容渲染
- ✅ CSS 样式覆写（适配阅读配置）

**关键依赖风险**：

| 原项目依赖 | Flutter 替代方案 | 风险等级 | 说明 |
|-----------|----------------|---------|------|
| `modules:book` 模块 | archive + xml | 🟡 中 | 需要自行实现 OPF/NCX 解析，原项目有成熟实现可参考 |
| Android HTML 渲染 | flutter_widget_from_html | 🔴 **高** | **EPUB 渲染是全书格式支持中最复杂的**。原版 Android 使用 WebView 或自定义 View 渲染 HTML，Flutter 的 HTML 渲染方案（flutter_html、flutter_widget_from_html）对 CSS 的支持有限，复杂的 EPUB 书籍排版可能错乱 |
| CSS 字体嵌入 | 自定义处理 | 🟡 中 | EPUB 内嵌字体需要提取并注册到 Flutter |
| 防复制 EPUB | 自定义解密 | 🟡 中 | 部分 EPUB 有 DRM 加密，无法解析 |

**降级方案**：若 HTML 渲染效果差，可使用 `webview_flutter` 作为降级方案，通过 WebView 渲染 EPUB 内容。
